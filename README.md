[ 🛠 КАК УСТАНОВИТЬ ПЛАГИН (ИНСТРУКЦИЯ) ]
1. Скачай файл: Сначала скачай OmniAI_v1.0.rbxmx по ссылке из автовыдачи.

2. Зайди в Studio: Открой любой свой проект или создай новый Baseplate.

3. Открой папку плагинов:

 В верхнем меню выбери вкладку PLUGINS.

 Найди кнопку Plugins Folder.

4. Установи файл: В открывшуюся папку просто перетащи скачанный файл OmniAI.rbxmx.

5. Готово: Перезапусти Roblox Studio. Плагин появится в верхней панели!
